# Interface principale

## Vue d'ensemble du menu

Le menu principal est ton point de départ à chaque lancement du jeu. Il donne accès à toutes les fonctionnalités en un coup d'œil.

---

## Les boutons du menu

### ▶️ Jouer
Lance ta partie. Tu seras dirigé vers l'écran de sélection des mini-jeux, où tu pourras choisir à quoi jouer selon ton niveau de progression.

### 🌐 Langue
Ouvre la liste des langues disponibles. Clique sur une langue pour l'appliquer immédiatement. Le jeu n'a pas besoin d'être redémarré.

### 🎨 Thème
Ouvre le sélecteur de thèmes visuels. Chaque thème change les images, les couleurs et les sons du jeu. Fais défiler pour voir tous les thèmes disponibles.

### 🐛 Signaler un bug
Ouvre le formulaire de rapport de bug. Tu peux y décrire le problème, joindre des captures d'écran et envoyer le tout directement à l'équipe.

### ❌ Quitter
Ferme le jeu. Ta progression est sauvegardée automatiquement — pas besoin de faire quoi que ce soit.

---

!!! tip "Raccourcis clavier"
    Tu peux naviguer dans les menus avec les touches fléchées et valider avec `Entrée`.
