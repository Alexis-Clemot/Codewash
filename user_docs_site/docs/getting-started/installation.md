# Installation

## Ce dont tu as besoin

Avant de commencer, vérifie que tu as bien :

- Un PC sous **Windows 10 ou 11**
- Au moins **500 Mo d'espace libre** sur ton disque
- Une connexion internet pour le premier lancement

---

## Étape 1 — Télécharger le jeu

1. Va sur la page de releases du projet sur GitHub
2. Clique sur la dernière version disponible (ex : `v1.2.0`)
3. Télécharge le fichier `CodeWash-setup.exe`

!!! warning "Attention"
    Télécharge toujours depuis la page officielle GitHub. Ne fais pas confiance à d'autres sources.

---

## Étape 2 — Lancer l'installation

1. Double-clique sur `CodeWash-setup.exe`
2. Si Windows affiche un avertissement de sécurité, clique sur **"Informations complémentaires"** puis **"Exécuter quand même"**
3. Suis les étapes de l'installateur
4. Clique sur **Terminer** une fois l'installation terminée

---

## Étape 3 — Lancer le jeu

1. Un raccourci **CodeWash** a été créé sur ton bureau
2. Double-clique dessus pour lancer le jeu
3. Un écran de chargement apparaît — c'est normal, le jeu se prépare

!!! tip "Le jeu ne se lance pas ?"
    Consulte la section [Problèmes fréquents](../troubleshooting/faq.md) si tu rencontres une erreur au lancement.

---

## Désinstaller le jeu

Si tu veux désinstaller CodeWash :

1. Va dans **Paramètres Windows → Applications**
2. Cherche **CodeWash** dans la liste
3. Clique sur **Désinstaller**

Tes sauvegardes sont conservées dans `Documents/CodeWash/saves/` et ne seront pas supprimées automatiquement.
