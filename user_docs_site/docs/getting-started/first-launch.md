# Premier lancement

## L'écran de chargement

Au premier lancement, un **écran de chargement** apparaît avec une barre de progression. C'est normal — le jeu initialise ses ressources. Ça ne prend que quelques secondes.

---

## Le menu principal

Une fois chargé, tu arrives sur le **menu principal**. Voici ce que tu y trouves :

| Bouton | Ce que ça fait |
|---|---|
| **Jouer** | Lance une partie |
| **Langue** | Change la langue du jeu |
| **Thème** | Change l'apparence visuelle |
| **Signaler un bug** | Envoie un rapport de bug |
| **Quitter** | Ferme le jeu |

---

## Créer ta partie

1. Clique sur **Jouer**
2. Si c'est ton premier lancement, une nouvelle partie est créée automatiquement
3. Tu arrives sur l'écran de sélection des mini-jeux

!!! info "Pas de compte nécessaire"
    CodeWash ne nécessite pas de créer un compte. Ta progression est sauvegardée localement sur ton PC.

---

## Reprendre une partie existante

Tes sauvegardes sont chargées automatiquement. Si tu avais déjà joué, tu retrouveras ta progression, tes succès, et tes collectables dès le lancement.
