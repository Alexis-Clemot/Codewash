# Changer le thème

Les **thèmes** changent l'apparence complète du jeu : images, couleurs, sons. C'est le moyen le plus rapide de donner un nouveau look à ton expérience de jeu.

---

## Comment changer le thème ?

1. Depuis le **menu principal**, clique sur **Thème** 🎨
2. Le sélecteur de thèmes s'ouvre avec un aperçu de chaque thème
3. **Fais défiler** pour voir tous les thèmes disponibles
4. Clique sur un thème pour l'appliquer immédiatement

---

## Thèmes disponibles

| Thème | Description |
|---|---|
| 🌿 **Thème par défaut** | Le thème original du jeu |
| 🚀 **Thème 2** | *(à compléter selon les thèmes du projet)* |
| 🌊 **Thème 3** | *(à compléter)* |
| *(d'autres à venir)* | 🔜 |

---

## Ce que change le thème

Quand tu changes de thème, voici ce qui est affecté :

- ✅ Les images du menu principal
- ✅ Les images dans les mini-jeux
- ✅ La musique de fond
- ✅ Les effets sonores
- ✅ Les couleurs de l'interface

!!! warning "Images déformées ?"
    Si des images semblent étirées ou déplacées après un changement de thème, c'est un bug connu. [Signale-le](bug-report.md) avec une capture d'écran.

---

## Revenir au thème par défaut

Ouvre le sélecteur de thèmes et clique sur **Thème par défaut**. Le jeu retrouve son apparence originale instantanément.
