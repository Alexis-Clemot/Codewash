# Changer la langue

CodeWash est disponible en plusieurs langues. Tu peux changer la langue à tout moment depuis le menu principal.

---

## Comment changer la langue ?

1. Depuis le **menu principal**, clique sur le bouton **Langue** 🌐
2. Une liste des langues disponibles s'affiche
3. Clique sur la langue de ton choix
4. Le jeu se met à jour immédiatement — pas besoin de redémarrer

---

## Langues disponibles

| Langue | Disponible |
|---|---|
| 🇫🇷 Français | ✅ Oui |
| 🇬🇧 Anglais | ✅ Oui |
| *(d'autres à venir)* | 🔜 Bientôt |

---

!!! info "La langue change tout le jeu"
    Les menus, les mini-jeux, les descriptions de succès — tout est traduit. Si tu remarques un texte qui n'est pas traduit, c'est un bug : n'hésite pas à le [signaler](bug-report.md).

---

## La langue n'a pas changé ?

Si après avoir cliqué sur une langue rien ne change :

- Ferme et relance le jeu
- Si le problème persiste, [signale le bug](bug-report.md)
