# Signaler un bug

Si quelque chose ne fonctionne pas correctement dans CodeWash, tu peux nous l'envoyer directement depuis le jeu. C'est rapide et ça nous aide vraiment à améliorer le jeu.

---

## Comment signaler un bug ?

1. Depuis le **menu principal**, clique sur **Signaler un bug** 🐛
2. Le formulaire de rapport s'ouvre
3. **Décris le problème** : qu'est-ce qui s'est passé ? À quel moment ?
4. **Ajoute des fichiers** si tu le souhaites (capture d'écran, logs...)
5. Clique sur **Envoyer**

Un message de confirmation s'affiche quand le rapport a bien été reçu.

---

## Qu'est-ce qu'un bon rapport de bug ?

Plus ton rapport est précis, plus on peut t'aider vite. Voici ce qui est utile à mentionner :

!!! tip "Un bon rapport contient..."
    - Ce que tu faisais quand le bug s'est produit
    - Ce que tu t'attendais à voir
    - Ce qui s'est passé à la place
    - Si le bug se reproduit à chaque fois ou seulement parfois

**Exemple de bon rapport :**
> *"J'ai cliqué sur le thème 'Space' et les images du mini-jeu de craft n'ont pas changé — elles restaient sur l'ancien thème. Ça se reproduit à chaque fois."*

---

## Joindre des fichiers

Tu peux joindre :

- 📸 **Des captures d'écran** — pour montrer le problème visuellement
- 📄 **Les logs du jeu** — générés automatiquement dans `Documents/CodeWash/logs/`
- 💾 **Ton fichier de sauvegarde** — si le bug concerne ta progression

---

## Le bouton ne fonctionne pas ?

Si le bouton de signalement de bug ne répond pas, tu peux contacter l'équipe directement par email à l'adresse configurée dans les paramètres du jeu.
