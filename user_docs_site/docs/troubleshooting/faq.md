# Problèmes fréquents

## 🚀 Lancement

??? question "Le jeu ne se lance pas"
    Vérifie que tu es bien sur Windows 10 ou 11. Si l'icône du bureau ne répond pas, essaie de relancer depuis le fichier `CodeWash.exe` directement dans le dossier d'installation.

    Si une erreur s'affiche, note le message et [signale le bug](../features/bug-report.md).

??? question "L'écran de chargement tourne indéfiniment"
    - Attends 30 secondes — le premier lancement peut être un peu plus long
    - Si rien ne se passe après ça, ferme le jeu et relance-le
    - Si le problème persiste, supprime le dossier `Documents/CodeWash/cache/` et relance

??? question "Le jeu s'ouvre mais l'écran est noir"
    Essaie de passer le jeu en mode fenêtré : maintiens `Alt` et appuie sur `Entrée` au lancement.

---

## 🎮 Gameplay

??? question "Ma progression a disparu"
    Tes sauvegardes sont dans `Documents/CodeWash/saves/`. Vérifie que le dossier existe et qu'il contient bien des fichiers. Si le dossier est vide, ta sauvegarde a peut-être été supprimée par erreur — on ne peut malheureusement pas la récupérer.

??? question "Un mini-jeu est verrouillé alors que j'ai le bon niveau"
    Ferme et relance le jeu. Si le verrou persiste, [signale le bug](../features/bug-report.md) en précisant ton niveau et le mini-jeu concerné.

??? question "Le jeu rame ou est lent"
    - Ferme les autres applications ouvertes sur ton PC
    - Vérifie que ton PC n'est pas en mode économie d'énergie
    - Redémarre le jeu

---

## 🎨 Thèmes et langue

??? question "Après un changement de thème, certaines images n'ont pas changé"
    C'est un bug connu sur certains mini-jeux. [Signale-le](../features/bug-report.md) avec une capture d'écran — ça nous aide à corriger le problème.

??? question "La langue n'a pas changé après avoir cliqué dessus"
    Ferme et relance le jeu. Si ça ne change toujours pas, [signale le bug](../features/bug-report.md).

---

## 🐛 Signalement

??? question "Le bouton 'Signaler un bug' ne répond pas"
    Contacte directement l'équipe par email. L'adresse de contact est disponible dans les paramètres du jeu ou sur la page GitHub du projet.

---

!!! question "Tu n'as pas trouvé ta réponse ?"
    [Signale ton problème via le formulaire de bug](../features/bug-report.md) — l'équipe te répondra dès que possible.
