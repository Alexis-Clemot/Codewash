# CodeWash 🎮

Application desktop éducative — **Electron + React + Vite**.  
Projet CodeWash — Epitech GING400.

---

## Installation et lancement

**Prérequis :** Node.js 18+ et npm 9+
```bash
git clone https://github.com/Alexis-Clemot/Codewash.git
cd Codewash/project/project
npm install --ignore-scripts
npm run dev
# → http://localhost:5173
```

**Mode Electron (application desktop)**
```bash
npm run electron:dev
```

**Build exécutable Windows**
```bash
npm run electron:build
# → release/CodeWash-Portable.exe
```
